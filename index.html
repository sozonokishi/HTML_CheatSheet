HTML Positioning Cheatsheet

Positioning

The position CSS property determines how an element positions in a HTML document. With top, right, bottom, left affecting the acts as an offset to determine the final position of an element.

Table Tags

Tag Description
position Affects the property of the positioning of an element. Accepted values includes static, relative, absolute, sticky
top Moves element downward from the top by set values.
right Moves element right from left by set values.
bottom Moves element upwards from the bottom by set values.
left Moves element left from the right by set values.

Table position Properties

Attribute Description
static The default attribute for elements. Element is positioned based on the normal flow of the document.
relative Element is positioned in relative on the normal flow of the document, where it is offset by the stated value from top, right, bottom, left.
absolute Element is removed from the normal flow of the document and is instead positioned based on the start of the document, where it's final position is determinated by top, right, bottom, left.
sticky Element behaves similar to static, and then offset relative to the scrolling position of the user, with the position determined by top, right, bottom, left.
